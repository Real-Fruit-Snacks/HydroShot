NAME

hydroshot - Fast, lightweight screenshot capture and annotation tool built with Rust.

SYNOPSIS

hydroshot [--region | --window] [--delay SECONDS]

DESCRIPTION

hydroshot is a native screenshot tool in pure Rust. It features region capture by drag, window capture by click, timed capture with 3/5/10-second countdown, multi-monitor coverage with a fullscreen overlay that dims inactive areas. Captured shots flow into an annotation surface with 14 tools backed by command-pattern undo/redo. Pin captures as always-on-top reference windows; copy to clipboard, save to file, upload anonymously to Imgur, or extract text via the Windows OCR API.

No Electron, no browser engine — just winit for windowing and tiny-skia for 2D drawing. Configuration lives in TOML. Tray-resident with a global hotkey (Ctrl+Shift+S by default).

OPTIONS

-r, --region
Start in region selection mode (default).
-w, --window
Start in window selection mode.
-d, --delay SECONDS
Delay the capture by the specified number of seconds.

EXAMPLES

To clone the repository and build the binary:

git clone https://github.com/Real-Fruit-Snacks/HydroShot.git
cd HydroShot
cargo build --release
