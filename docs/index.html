Features

Everything you need to capture and annotate

HydroShot combines fast screen capture with a full suite of annotation tools, all wrapped in a beautiful Catppuccin Mocha interface.

Region & Window Capture

Click and drag for precise region selection, or click any window to capture it. Multi-monitor support captures your entire virtual desktop.

14 Annotation Tools

Arrows, rectangles, rounded rects, circles, lines, pencil, highlighter, spotlight, text, pixelation, step markers, eyedropper, and measurement. Adjust size with the scroll wheel.

Flexible Export

Copy to clipboard, save to file, upload to Imgur, extract text with OCR, or pin captures as floating windows. Recent captures history for quick re-access.

Catppuccin Theming

Beautiful Catppuccin Mocha dark theme with 5 color presets. Right-click any color for a native color picker. Customizable keyboard shortcuts and configurable toolbar.

System Integration

System tray with left-click capture, global hotkey, auto-start on login, capture history with thumbnails, and a full CLI for automation.

Delay & Countdown

Timed captures with 3s, 5s, or 10s delays and a visible countdown overlay. Perfect for capturing menus, tooltips, and transient UI.

Command Line

Automate with the CLI

Script captures and integrate HydroShot into your workflow.

hydroshot
# Capture and copy to clipboard
$ hydroshot capture --clipboard

# Save to a file
$ hydroshot capture --save

# Delayed capture (3 seconds)
$ hydroshot capture --delay 3

# Combine options
$ hydroshot capture --delay 5 --clipboard
